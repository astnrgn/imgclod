# imgclod

## env variables
Create a ```.env``` file with:

```
VUE_APP_CLOUDINARY_UPLOAD_PRESET=
VUE_APP_CLOUDINARY_URL=
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
